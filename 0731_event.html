이벤트

  • 이벤트 : 웹 브라우저나 사용자가 행하는 동작
  • 웹 문서(DOM) 영역 안에서 발생하는 동작
  • 마우스, 키보드 , 웹 문서 로딩시, 폼 내용 작성 ,,,(폼, li, 메뉴영역)
    • html에서 가상클래스로 사용했던것들
    • 사용자의 동작과 관련
    • (폼, li, 메뉴영역)

이벤트의 종류

  • 마우스 관련 이벤트 요소
    • onclick : HTML 요소, 영역을 클릭할 때 발생
    • ondbclick : 두번 클릳릭할 때 발생
    • onmouseover :
    • onmouseout :
    • onmousemove :
    • onmousedown :
    • onmouseup :
    • onwheel :
  • 키보드 이벤트
    • onkeypress : 키를 누를 때
    • onkeydown : 키를 누르는 동안
    • onkeyup : 눌렀던 키가 올라올때,(손을 땔 때)
    • keypress,keydown 구분이 어려워 보통 keyup을 많이 사용
  • 포커스 이벤트
    • onfocus : 요소에 포커스(커서,포인터) 가 되면
    • onblur : 요소가 포커스를 잃으면
  • 폼 이벤트
    • onsubmit : 폼 적송 될 때
    • reset : 폼 요소값의 초기화
  • 문서 관련 이벤트
    • onlode : 문서 로딩이 끝나면
    • resize : 문서 화면의 크기각 발생
    • scroll : 화면이 스크롤 되면 발생
    • aboart : 문서를 완전히 로딩되기 전에 멈춘다
    • error : 문서가 정획히 로딩되지 않고 에러가 발생한 경우
    • unlaode : 문서를 벗어날 때 발생
