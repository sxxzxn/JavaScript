• for문

    for ( 초기값; 조건; 증가식){ 실행문; 조건문; 반복문; 함수; 선언문; ..... }

    for(i=0; true;){ }

    for(;;){ ~~무한루프~~ }

  • 초기값 : for문에서 사용하는 반복 조건값을 초기화
  • 조건 : 조건이 true 동안에 for문 반복
  • 증가식 : for 문이 반복되는 동안 초기값을 증가 시킴
  • while문

  • while(조건){
    실행구문; for문; while문; 함수; 선언; break; continue; ...}

  • do~while문

  • do{
    실행구문 break; continue; } while(조건)
