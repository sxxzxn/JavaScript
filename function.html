함수(function)

  • 어떤 목적을 가지고 작성한 코드를 모아놓은 블록문
  • 각 명령의 시작과 끝을 정확히 구분할 수 있음
  • 함수를 정의한다 : 블록문을 구성한다
    • 블록문을 구성한다
    • function키워드
    • 식별자(변수)
    • 소괄호
  • 코드의 재활용이 가능
  • function 함수명(매개변수){ 실행구문; 함수; 로직; 변수; [return값;] }
  • 함수표현식
    • 익명 함수
      • 변수명에 할당해 변수를 함수처럼 사용
      • let 변수명 function (매개변수){}
      • 변수명 ( 매개변수,,,,)
      • 변수명 (10,20)
      • function sum(매개변수1, 매개변수2) { }, sum(10,20)
      • inline
    • 네이밍 함수
      • 익명함수와 유사
      • 변수에 할당하는 함수에 식별자가 있을 때
      • const 변수명= function 이름(){};
      • 변수명으로만 참조 가능
    • 즉시 실행 함수
      • (function(){ 실행구문; }()); : 기본형
      • (function(매개변수,,,,){ 실행구문,,, }(인수)); : 확장형
        • 인수안에 이벤트 넣어주기
        • 페이지가 실행될때 자동으로 즉시 실행
    • 화살표 함수
      • ES6 정의
      • ()=>{}
      • const sum = ()=>{로직}; sum();

      • onclick="(e)=>{e.taget...}" 이런식으로 사용

      • 클릭 한번에만 사용!
