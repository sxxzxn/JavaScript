자바스크립트의 자료형

  1. 숫자형
    • 숫자 형태의 데이터를 처리
    • 정수 : -3, -2, 0, 1, 10
    • 실수 : -0.25, 0, 3.0
    • 실수의 소숫점 이하 자릿수에 대하여 정밀도를 보장하지 않음
  2. 문자열(string)
    • 문자가 한 개 이상 이어져 있는 것을 문자열
    • 'a', "a", 'aaaa', "aaaa"
  3. 논리형(boolen)
    • true false 값 표현
    • 조건을 확인하여 맞으면 true, 틀리면 false
  4. undefined유형
    • 자료형이 정의되지 않았을 때의 데이터 상태
    • 변수 선언만 하고 값이 할당되지 않은 상태
  5. null유형
    • 데이터 값이 유효하지 않은 상태
    • 변수에 할당된 값이 유효하지 않다는 의미
  6. 배열(array)
    • 하나의 변수에 여러 값을 저장 할 수 있음
    • 자료형의 유형이 같은 값을 할당
  7. 객체(object)
    • 배열도 객체의 일종
    • 함수와 속성을 함께 표시
    • 배열, 함수, 객체 리터럴 등으로 파생되는 상위 자료형
    • 객체 리터럴 : 중괄호({}) 를 사용, 키(key), 값(value) 한 쌍으로 이루어진 데이터
  8. var name;
  9. 연산자의 종류
    • +, -, *, / : 수학과 동일
    • % : 퍼센트 연산자, 모듈러 연산자 - 나머지 연산자
    • ++ : 피연산자를 1증가
    • ** : 제곱승 ==> a**a
    • 삼항연산자
      • <% (조건1) ? a:b%> 항목1
      • 항목2
